# Leo Real Estate Landing Page

Files:
- `index.html` → page structure
- `styles.css` → layout, motion, look
- `content.js` → easiest place to swap words, contact info, and placeholders later
- `script.js` → fills the page with content and handles animation

Fastest way to edit later:
1. Open `content.js`
2. Change the text, phone, email, reviews, and service blurbs
3. Replace the hero photo area in `index.html` when you’re ready with a real image

To preview locally:
```bash
cd real-estate-site
python3 -m http.server 4173
```
Then open:
`http://localhost:4173`
